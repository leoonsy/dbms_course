DBMS_Course
=============================

Программа для управления базой данных компьютерного магазина. Курсовая работа. Используется библиотека fancygrid.js.

Установка
------------

Измените конфигурацию в файле core/config.php. Выполните в MySQL файл dbms_course.sql.
